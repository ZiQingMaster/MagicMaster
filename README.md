# Magic-Master 项目介绍
Magic是子晴自主开发的一个Hacked Client For Minecraft客户端
本项目只供参考  目前正在正常开发 非Dev团队切勿商用
本客户端基于MCP918 后续会更换版本开发Fabric版本
# Magic-Master 如何构建/启动
需安装JDK 8
需安装Git
需安装IntelliJ IDEA
需下载MCP918
反编译MCP918
克隆这个仓库至Src
打开IntelliJ IDEA
设置启动配置(启动类为Start，在VM参数中添加natives的目录)-Djava.library.path=你的natives目录
导入lib包
启动客户端
